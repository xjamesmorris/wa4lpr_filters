This repo contains Pascal source code and PDF documentation provided by Dennis Sweeney WA4LPR. It is published here on Github with his permission.

- INTRFIL: a program intended to create bandpass filters using interdigitated round
rod Â¼ wavelength long resonators between a parallel ground planes (slab line).

- WGFIL: a program that will design a wide variety of post and aperture coupled
bandpass filters in rectangular waveguide.

See the respective directories for documentation.

These applications may be compiled under Linux with the 'fpc' compliler, e.g.:

`$ fpc INTRFIL_Ver_1-0.pas`

You can find more information on waveguide filters and an example of using WGFIL in Paul Wade (W1GHZ)'s QEX articles:
- [Waveguide Filters You Can Build - and Tune Part 1 - Filter Tour](https://www.w1ghz.org/filter/QEX_Nov_2009_p40-43.pdf)
- [https://www.w1ghz.org/filter/QEX_Jan_2010_p20-25.pdf](https://www.w1ghz.org/filter/QEX_Jan_2010_p20-25.pdf)
